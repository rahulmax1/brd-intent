# Adapt Canvas

## Design Context

### Users
Mixed audience — security teams, clients, and project managers collaborating on penetration testing scope and requirements. They're reviewing a structured intent model (actors, entities, journeys, rules) to reach consensus before testing begins. Mindset is analytical and detail-oriented, but they need clarity not complexity. The AI chat panel helps them iterate on the model without leaving the review context.

### Brand Personality
**Smart, clean, modern** — like Linear. Professional without being corporate. Technical without being intimidating. The tool should feel like a well-crafted instrument that gets out of the way.

### Aesthetic Direction
- **Visual tone**: Minimal, precise, warm neutrals with cool blue accents
- **Reference**: Linear — clean, fast, keyboard-first, purposeful use of color
- **Anti-references**: Jira (dense/overwhelming), generic Bootstrap (no personality), overly playful/gamified (not taken seriously), dark/hacker aesthetic (intimidating)
- **Theme**: Light mode primary. Warm off-white (#F8F8F7) backgrounds, navy brand accent, blue (#0081F2) interactive accent
- **Font**: DM Sans Variable — modern geometric sans-serif

### Design Principles
1. **Clarity over decoration** — every element earns its place. Status colors, not ornamental borders. Hierarchy through weight and spacing, not visual noise.
2. **Context without clutter** — show what matters for the current task. Collapsible sections, scoped editing, progressive disclosure. Never dump everything at once.
3. **Warm precision** — professional but not cold. Warm neutrals, generous whitespace, soft shadows. The interface should feel calm and trustworthy.
4. **AI as collaborator, not magic** — the chat panel is a working tool, not a novelty. Show exactly what changed (diffs), let users approve/reject, maintain full version history. Transparency builds trust.
5. **Consistent rhythm** — uniform spacing scale, predictable interactions, 200ms transitions. The interface should feel like one cohesive system, not a patchwork of components.
